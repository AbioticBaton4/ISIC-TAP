# Tópicos Avanzados de Programación 

Este repositorio contiene el material, ejercicios y proyectos correspondientes a la materia de **Tópicos Avanzados de Programación**. El curso está diseñado para profundizar en el desarrollo de software robusto, interfaces interactivas y arquitecturas modernas.

## Descripción del Curso
La asignatura de Tópicos Avanzados de Programación proporciona al estudiante las competencias necesarias para desarrollar aplicaciones que involucren interfaces gráficas de usuario, el uso de librerías externas, programación multihilos para mejorar el rendimiento, acceso eficiente a bases de datos y una introducción al desarrollo móvil.

---

## Temario

### Unidad 1: Interfaz Gráfica de Usuario (GUI)
* 1.1 Creación de interfaz gráfica para usuarios.
* 1.2 Tipos de eventos.
* 1.3 Manejo de eventos.
* 1.4 Manejo de componentes gráficos de control.

### Unidad 2: Componentes y Librerías
* 2.1 Definición conceptual de componentes, paquetes y librerías.
* 2.2 Uso de librerías proporcionadas por el lenguaje.
* 2.3 Creación de componentes (visuales y no visuales) definidos por el usuario.
* 2.4 Creación y uso de paquetes/librerías definidas por el usuario.

### Unidad 3: Programación Concurrente (MultiHilos)
* 3.1 Concepto de hilo.
* 3.2 Comparación de un programa de flujo único contra uno de flujo múltiple.
* 3.3 Creación y control de hilos.
* 3.4 Sincronización de hilos de computación.

### Unidad 4: Acceso a Datos
* 4.1 Introducción.
* 4.2 Conexión a origen de datos.
* 4.3 Manipulación de datos.
* 4.4 Visualización de datos.

### Unidad 5: Programación de Dispositivos Móviles
* 5.1 Introducción a las tecnologías y herramientas móviles.
* 5.2 Clasificación y aplicaciones de los dispositivos móviles.
* 5.3 Entorno operativo de las aplicaciones móviles.
* 5.4 Desarrollo de aplicaciones móviles.
* 5.5 Aspectos de seguridad.

* **Base de Datos:** SQLite / MySQL / PostgreSQL.

## Instalación y Configuración


Sigue estos pasos para configurar el entorno de desarrollo local y poder trabajar con los ejercicios de la materia:

### 1. Requisitos Previos
Asegúrate de contar con las siguientes herramientas instaladas:
* **Java Development Kit (JDK):** Versión 17 o superior.
* **Git:** Para clonar y gestionar el repositorio.
* **IDE Recomendado:** IntelliJ IDEA, Eclipse o Visual Studio Code (con el "Extension Pack for Java").

### 2. Clonar el Repositorio
Para obtener una copia local de este curso, abre tu terminal y ejecuta:

```bash
git clone https://github.com/AbioticBaton4/ISIC-TAP.git
```
