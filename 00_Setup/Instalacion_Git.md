# Guía de Instalación de Git

Para colaborar en este repositorio y gestionar tus proyectos de **Tópicos Avanzados de Programación**, necesitas instalar Git en tu equipo local.

## 1. Descarga de Git
Puedes descargar el instalador oficial según tu sistema operativo:
* **Windows:** [git-scm.com/download/win](https://git-scm.com/download/win)
* **macOS:** [git-scm.com/download/mac](https://git-scm.com/download/mac) (O usa `brew install git`)
* **Linux:** [git-scm.com/download/linux](https://git-scm.com/download/linux)

## 2. Proceso de Instalación (Recomendaciones)
Al ejecutar el instalador en Windows, asegúrate de marcar las siguientes opciones para mayor comodidad:
1.  **Git Bash Here:** Permite abrir la terminal de Git haciendo clic derecho en cualquier carpeta.
2.  **Use Git from the Windows Command Prompt:** Para que Git funcione tanto en CMD como en PowerShell.
3.  **Default Editor:** Si no usas Vim, selecciona **Visual Studio Code** como editor por defecto.

## 3. Verificación de la Instalación
Abre una terminal (CMD, PowerShell o Git Bash) y ejecuta el siguiente comando para confirmar que Git está listo [00:22:03]:

```bash
git --version
